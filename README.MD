# CHAT-UI

### Responsive chat
Interfaz de chat-ui desarrollada con HTML y CSS puro. Uso de etiquetas semanticas HTML5. Estructura responsive funcional, uso de flexbox en distribucion principal.

### Caracteristicas
Lista de contactos con estado en linea/fuera de linea. Ventana de conversacion estilo mensajeria instantanea. Interfaz completamente responsiva desde 480px hasta pantallas grandes.
